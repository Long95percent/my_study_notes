## WinMain函数
- 是Windows桌面应用程序的入口函数，类似于控制台程序中的main函数
- 是基于Win32 API的图形界面（GUI）程序的起点
```
int _clrcall WinMain(
  [in]           HINSTANCE hInstance,
  [in,optional]  HINSTANCE hPrevInstance,
  [in]           LPSTR     lpCmdLine,
  [in]           int       nShowCmd
  );
```
- hInstance：当前应⽤程序实例的句柄，⽤于标识程序的内存空间和资源。 
- hPrevInstance：在早期的Windows版本中使⽤，现在已弃⽤，始终为NULL。
- hPrevInstance：在早期的Windows版本中使⽤，现在已弃⽤，始终为NULL。
- nCmdShow：控制窗⼝显示⽅式的参数，可以是ShowWindow函数的nCmdShow参数中指定的任何值
## 窗口类注册
连接程序入口和窗口创建的关键步骤，Windows需要知道窗口的基本属性和行为
```
ATOM RegisterClassExa(
[in] const WNDCLASSEXA *unnamedParaml
);
```
- 窗口类分为4类：WNDCLASSA,WNDCLASSW,WNDCLASSEXA,WNDCLASSEXW
- EX-拓展类，AW-字符指针类型
- 以WNDCLASSEXA为例
     - ![[Pasted image 20250911225318.png]]
     - 参数具体用法：
     - https://learn.microsoft.com/zh-cn/windows/win32/api/winuser/ns-winuser-wndclassexa 
- 使用WNDCLASSEX结构注册窗口类
     - 部分成员必须显示设置：
         - **cbSize**:必须设置为sizeof(WNDCLASSEX),用于标识结构体版本和大小。
         - **lpfnWndProc**：必须指定有效的窗⼝过程函数指针，⽤于处理窗⼝消息。NULL->⽆法接收任何消息。
         - **hInstance**:必须提供当前应⽤程序实例的句柄，标识窗⼝类的所有者。
         - **lpszClassName**:：必须提供唯⼀的字符串名称，作为窗⼝类的标识符。此名称将⽤于后续的 CreateWindowEx 调⽤。
    - 可选设置成员（系统提供默认值）：
         - **style**:类样式标志，未设置就无特殊行为，通常建议⾄少设置为CS_HREDRAW | CS_VREDRAW，确保窗⼝⼤⼩变化时正确重绘。 
         - **cbClsExtra和cbWndExtra**:默认0.仅在需要为类或者窗口实例附加额外数据时设置。
         - **hIcon**:大图标句柄，NULL->系统使用默认应用程序图标
         - **hCursor**:光标句柄，NULL->系统使用标准箭头光标。
         - **hbrBackground**:背景画刷句柄。未设置则窗口背景绘制行为依赖系统设置。常用默认值：(HBRUSH)(COLOR_WINDOW+1) （系统窗⼝颜⾊）、GetStockObject(WHITE_BRUSH) （⽩⾊背景）
         - **lpszMenuName**:菜单资源名称。若为NULL,窗口无默认菜单。
         - **hIconSm**:小图标句柄。若为NULL,系统可能会自动缩放hIcon或使用默认图标
    - 实验：创建窗口
         - ![[Pasted image 20250912215224.png]]
    ```
    #include<Windows.h>

int WINAPI WinMain(HINSTANCE hInstance, HINSTANCE hPrecInstance, LPSTR lpCmdLine, int nSnowCmd)
{
	WNDCLASSEXA wndcls;
	wndcls.cbSize = sizeof(WNDCLASSEXA);
	wndcls.cbClsExtra = 0;
	wndcls.cbWndExtra = 0;
	wndcls.hbrBackground = (HBRUSH)GetStockObject(WHITE_BRUSH);
	wndcls.hCursor = LoadCursor(NULL, IDC_CROSS);
	wndcls.hIcon = LoadIcon(NULL, IDI_ERROR);
	wndcls.hInstance = hInstance;
	wndcls.lpfnWndProc = NULL;
	wndcls.lpszClassName = "test";
	wndcls.lpszMenuName = NULL;
	wndcls.style = CS_HREDRAW | CS_VREDRAW;
	wndcls.hIconSm = LoadIcon(NULL, IDI_WARNING);

	if (RegisterClassExA(&wndcls)) {
		MessageBoxA(NULL, "窗口类创建成功！", "成功", MB_OK | MB_ICONINFORMATION);
	}
	else {
		MessageBoxA(NULL, "窗口类创建失败", "错误", MB_OK | MB_ICONERROR);
	}

	return 0;
}
## 窗口创建
- 创建窗口类是为了调用CreatWindows等函数准备。
- CrreatWindow函数大体分为两类，CreatWindows和CreatWindowsEx。EX支持更多拓展窗口样式
- 可继续分，A/W，A代表该函数的字符指针参数为char指针（对应ANSI/GBK编码），W->wchar_t指针（Unicode/UTF-16 BE编码）

## 一、这三条指令在干什么？

```asm
xorps   xmm1, xmm1     ; 把 xmm1 清零 → xmm1 = 0.0
comisd  xmm1, xmm0     ; 比较 0.0 和 xmm0 里的 double

```

核心是：

> **比较： 0.0 和 xmm0 的值，然后根据标志位决定跳不跳转**

重点在第三条：`ja`

---

## 二、`comisd` 是怎么设置标志位的？

`comisd xmm1, xmm0` 本质相当于在做：

```c
if (xmm1 > xmm0)   // 这里是 0.0 > xmm0 吗？
```

然后它会设置标志位（重点是这两个）：

|条件|CF|ZF|
|---|---|---|
|xmm1 > xmm0|0|0|
|xmm1 < xmm0|1|0|
|xmm1 == xmm0|0|1|
|任何一个是 NaN（无效比较）|1|1|

也就是说，它的结果由这个判断决定：

```c
0.0 ? xmm0
```

## 三、`ja` 到底在检查什么？

### `ja` = jump if **above**（无符号大于）

它的语义是：

```asm
if (CF == 0 && ZF == 0)
    跳转
```

也就是：

|CF|ZF|会不会跳|
|---|---|---|
|0|0|✅ 跳|
|1|0|❌ 不跳|
|0|1|❌ 不跳|
|1|1|❌ 不跳|

所以：

> 只要 **CF 或 ZF 有一个是 1，`ja` 就不会跳转**

只有在 **CF = 0 且 ZF = 0** 的时候才会跳。

---

## 四、你为什么「两个改为 1 也改变方向」？

你说：

> ZF 和 CF 一开始都为 0，我把改为了 1，确实改变方向了

那是因为：

```
原本： CF=0 + ZF=0   → 满足 ja → 会跳
你改： CF=1 + ZF=1   → 不满足 → 不跳
```

所以你相当于强行 **破坏了跳转条件**，程序路径自然就变了。

---

## 五、如果你只改 CF 会怎样？

我们来列清楚你所有可能的操作效果（非常重要）：

|你怎么改|实际标志位|ja 会不会跳|
|---|---|---|
|不改|0,0|✅ 跳|
|只改 CF=1|1,0|❌ 不跳|
|只改 ZF=1|0,1|❌ 不跳|
|CF=1, ZF=1|1,1|❌ 不跳|
|强制 CF=0, ZF=0|0,0|✅ 一定跳|

结论：

> **你只需要改其中一个位，就足以改变 ja 的结果。**

而且不管你怎么改，只要不是 `0,0`，都会「不跳」。

# 为什么要这么做？（动机）

这是一种经典的**自删除**（self-delete）实现方法：

1. 当前程序启动一个 `cmd` 进程；
    
2. `cmd` 先 `timeout`（等待一小段时间，保证原程序退出并释放文件句柄）；
    
3. 然后执行 `del "<exe path>"` 删除可执行文件；
    
4. 原程序自己退出（假设调用此函数后程序会退出），所以目标文件在等待后能被删除。
    

使用 `CREATE_NO_WINDOW` 是为了不弹出黑窗，`/NOBREAK` 防止用户按键打断 timeout。

```c
DWORD sub_7FF68CAEBBE0(void)
{
    CHAR Filename[272];
    CHAR CommandLine[528];
    STARTUPINFOA StartupInfo;
    PROCESS_INFORMATION ProcessInformation;

    // 获取当前可执行文件路径
    if (GetModuleFileNameA(NULL, Filename, sizeof(Filename)) == 0)
        return 0;

    // 创建命令行：等待 25 秒，然后删除可执行文件
    snprintf(CommandLine, sizeof(CommandLine),
             "cmd /c timeout /T %d /NOBREAK && del \"%s\"",
             25, Filename);

    // 初始化 STARTUPINFOA
    ZeroMemory(&StartupInfo, sizeof(StartupInfo));
    StartupInfo.cb = sizeof(StartupInfo);

    ZeroMemory(&ProcessInformation, sizeof(ProcessInformation));

    // 创建进程，隐藏窗口
    if (CreateProcessA(NULL, CommandLine, NULL, NULL, FALSE,
                       CREATE_NO_WINDOW, NULL, NULL,
                       &StartupInfo, &ProcessInformation))
    {
        CloseHandle(ProcessInformation.hProcess);
        return CloseHandle(ProcessInformation.hThread); // 返回非零表示 CloseHandle 成功
    }

    return 0; // 创建失败
}
```

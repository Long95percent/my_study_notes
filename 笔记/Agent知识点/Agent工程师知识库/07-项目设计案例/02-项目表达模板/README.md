# 项目表达模板

## 项目介绍模板

1. **背景**：为什么要做这个 Agent？解决什么业务问题？
2. **目标**：希望 Agent 完成哪些任务，成功标准是什么？
3. **架构**：模型层、编排层、工具层、记忆层、后端服务如何连接？
4. **难点**：稳定性、工具调用、记忆召回、权限、安全、成本、延迟。
5. **方案**：用了哪些设计取舍？为什么这样做？
6. **结果**：成功率、响应时间、人工介入率、错误率、成本等指标。
7. **复盘**：如果重做，会如何改进？

# 模型选择与成本权衡

## 选型维度

- 能力：推理、代码、工具调用、长上下文、多模态。
- 成本：输入/输出 Token 单价、缓存命中、批处理。
- 延迟：首 token 延迟、总生成时间、并发吞吐。
- 稳定性：结构化输出成功率、幻觉率、拒答率。
- 部署：闭源 API、私有化、本地模型、合规要求。

## 常见策略

- 主模型负责复杂规划，小模型负责分类、抽取、摘要。
- 高风险工具调用前用校验模型或规则引擎复核。
- 对重复上下文启用缓存，对长文档先摘要再检索。

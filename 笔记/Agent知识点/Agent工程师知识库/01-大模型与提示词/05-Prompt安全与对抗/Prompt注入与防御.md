# Prompt 注入与防御

## 一句话定位

Prompt 注入是用户或外部文档试图覆盖系统指令、诱导模型泄露信息或越权调用工具。

## 常见攻击面

- 用户直接说“忽略之前的指令”。
- RAG 文档里藏恶意指令。
- 工具返回内容中包含诱导性文本。
- 多 Agent 之间传递未清洗上下文。

## 防御角度

- 指令分层：系统指令 > 开发者指令 > 用户输入 > 外部文档。
- 内容隔离：用 XML/Markdown 区分“数据”和“指令”。
- 工具白名单：模型不能自己决定越权工具。
- 参数校验：敏感操作必须二次确认。
- 输出审计：记录 prompt、工具调用、返回值和审批链路。

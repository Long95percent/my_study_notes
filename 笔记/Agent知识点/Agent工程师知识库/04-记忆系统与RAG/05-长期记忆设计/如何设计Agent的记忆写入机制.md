抖音@AI大模型-鹏宇
设置记忆提取层，在用户对话时在旁边默默观察
分为Backend Model:专门的模型分析模型，不占用主对话资源。Small Model（8B小模型）：处理实体提取、分类、降低成本与延迟

记忆准入判断：
- 持久性价值
     - 事实性信息：长期保存，比如会议时间
     - 临时推理：放入短期记忆并丢弃
- 结构化程度：
     - 高结构化：实体与关系（Entity&Relation）
     - 低结构化：纯文本对话，非结构化语料
- 个性化佳值：
     - 用户画像：习惯，偏好，编号
     - 隐藏需求：隐藏意图

**混合存储架构（Hybrid Storage）**
如果一股脑丢进去然后向量检索，容易有丢失和幻觉
需要根据数据结构类型，分流存储
- 向量数据库
- 图数据库
- 关系型DB
**动态维护机制**
- 评分系统：重要性加权（比如金钱、合约）
- 反思：定期总结深层洞察
- 遗忘/衰减：分级存储

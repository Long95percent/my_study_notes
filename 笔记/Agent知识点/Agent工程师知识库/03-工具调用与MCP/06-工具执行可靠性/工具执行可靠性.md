# 工具执行可靠性

## 一句话定位

工具调用不是“调一下 API”这么简单，生产环境必须处理超时、失败、重试、幂等、限流和副作用。

## 工程机制

| 机制 | 解决问题 | Agent 场景 |
|---|---|---|
| Timeout | 防止工具卡死 | 搜索、爬虫、数据库查询 |
| Retry with Backoff | 临时失败恢复 | API 429/5xx、网络抖动 |
| Circuit Breaker | 外部服务异常时快速失败 | 第三方工具不可用 |
| Idempotency Key | 防止重复副作用 | 支付、发邮件、写数据库 |
| Rate Limit | 防止成本失控/被封禁 | 搜索、模型调用、外部 API |
| Sandbox | 隔离危险代码执行 | 代码解释器、文件操作 |

# 成本延迟与容量规划

## 成本来源

- 模型输入输出 Token。
- Embedding 批量生成。
- Reranker 调用。
- 向量库、搜索引擎、数据库资源。
- 工具调用的第三方 API 费用。

## 延迟来源

- 模型首 token 延迟。
- 多轮工具调用串行等待。
- RAG 检索和重排。
- 外部 API 超时。

## 优化策略

- 缓存：Prompt cache、工具结果缓存、检索结果缓存。
- 并行：独立工具并发调用。
- 分层模型：小模型做轻任务，大模型做复杂推理。
- 限流：按用户、租户、工具维度限流。

# 冲突解决与结果汇总

## 一句话定位

多个 Agent 输出不一致时，需要汇总、裁决和可解释的取舍机制。

## 常见策略

- Supervisor 汇总：主控 Agent 统一裁决。
- Critic 评审：独立 Agent 检查事实、逻辑、格式。
- 投票机制：多个候选答案多数决或加权评分。
- 证据优先：以工具返回、数据库记录、检索证据为准。
- 人工审批：高风险分歧交给人。

## 风险

- 多 Agent 互相说服但都没证据。
- 上下文膨胀导致成本增加。
- 循环讨论不收敛。

# 错误恢复与重规划

## 一句话定位

Agent 不可能每一步都成功，工程重点是失败后能识别、恢复、降级或请求人工帮助。

## 常见失败

- 工具调用参数错误。
- 外部 API 超时或限流。
- 模型输出 JSON 不合法。
- 计划步骤过期或前置条件不满足。
- 检索结果不相关。

## 处理策略

- 可重试：网络抖动、临时超时。
- 可修复：JSON 格式错误、参数类型错误。
- 可重规划：工具不可用、路径走不通。
- 需人工：高风险操作、权限不足、数据不确定。

# Human-in-the-loop 审批

## 一句话定位

高风险或不确定操作不要让 Agent 自动执行，而是在关键节点引入人工审批。

## 适合审批的动作

- 发邮件、转账、删除数据、提交代码、调用付费 API。
- 访问敏感信息或跨权限数据。
- 模型信心低、检索证据不足的任务。

## 设计点

- 审批内容：动作、参数、原因、影响范围、回滚方案。
- 审批结果：同意、拒绝、修改参数、要求补充信息。
- 审计：谁审批、何时审批、审批依据是什么。

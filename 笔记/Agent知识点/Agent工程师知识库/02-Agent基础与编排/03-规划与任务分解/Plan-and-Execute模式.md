# Plan-and-Execute 模式

## 一句话定位

Plan-and-Execute 是先生成任务计划，再按步骤执行，适合更长、更复杂的任务。

## 适合场景

- 多步骤任务：调研、代码修改、数据处理、报告生成。
- 需要任务拆分和进度追踪。
- 需要人工审批计划后再执行。

## 工程关注

- 计划粒度不能太粗，也不能太碎。
- 每步要有输入、输出、完成条件。
- 执行失败后需要重规划。

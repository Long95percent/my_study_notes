# 部署与运行环境

## Agent 系统部署关注点

- 模型 API 网络连通性和超时。
- 工具运行时隔离：容器、沙箱、权限。
- 长任务：后台 worker、队列、状态持久化。
- 横向扩展：无状态 API + 有状态存储。
- 回滚：工具版本、Prompt 版本、模型版本都要可追踪。

## 技术选择

- Docker：本地和服务器部署一致性。
- Kubernetes：多服务、弹性伸缩、服务发现。
- Serverless：轻量事件触发任务，冷启动需注意。

## 中优先级：工程深度加分

- [[队列与事件驱动]]：异步任务、长任务、削峰填谷。
- [[Tracing与可观测性]]：排查 Agent 多步骤执行问题。
- [[成本延迟与容量规划]]：Token 成本、缓存、并发控制。
- [[工具执行可靠性]]：超时、重试、熔断、幂等。
- [[数据摄取与索引管线]]：从原始文档到可检索索引。

## 数据库与存储专题

- [[PostgreSQL]]：结构化业务数据、事务、权限、pgvector。
- [[SQLite]]：本地轻量状态、单机原型、边缘端。
- [[Redis]]：缓存、短期记忆、分布式锁、队列。
- [[Milvus和FAISS]]：向量检索基础。
- [[Qdrant]]：向量库、Payload Filter、工程化部署。
- [[Weaviate]]：向量库 + Schema + 混合检索。
- [[Elasticsearch与OpenSearch]]：全文检索、BM25、日志搜索。
- [[Neo4j]]：知识图谱、实体关系、多跳推理。
- [[ClickHouse]]：日志分析、埋点、评测数据聚合。
- [[S3与MinIO]]：原始文件、切片文件、离线数据集。

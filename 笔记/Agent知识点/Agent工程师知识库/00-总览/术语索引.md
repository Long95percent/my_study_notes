# 术语索引

## Agent 核心

- ReAct：Reason + Act，模型先思考再调用工具，观察结果后继续。
- Plan-and-Execute：先生成计划，再逐步执行。
- Reflection：执行后自我检查或让 Critic Agent 评审。
- Checkpoint：保存状态，便于恢复长任务。
- Human-in-the-loop：人工审批或人工纠偏。

## RAG 与记忆

- Chunking：把长文档切成可检索片段。
- Embedding：把文本映射成向量。
- ANN：Approximate Nearest Neighbor，近似最近邻检索。
- BM25：传统全文检索打分算法。
- Hybrid Search：向量检索 + 关键词检索混合。
- Reranker：对召回结果重新排序。
- Recall@K：前 K 个结果是否召回目标答案。
- Faithfulness：回答是否忠实于检索材料。

## 工具与协议

- Tool Calling：模型输出工具名和参数。
- ToolRegistry：工具注册中心。
- ToolRuntime：工具执行环境。
- MCP：Model Context Protocol，模型上下文协议。
- JSON Schema：参数和输出结构校验规范。

## 工程化

- Idempotency：幂等，重复执行不会造成额外副作用。
- Circuit Breaker：熔断，外部服务异常时快速失败。
- Rate Limit：限流。
- Retry with Backoff：指数退避重试。
- Tracing：跨步骤链路追踪。
- PII：个人敏感信息。
